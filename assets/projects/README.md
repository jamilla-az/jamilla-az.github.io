# Project images

Drop images for blog posts here. For the Spotify recommender post, use:

- `spotify-intro.png` — project overview or dataset snapshot
- `spotify-eda.png` — EDA or dataset statistics
- `spotify-model.png` — schematic of matrix factorization or pipeline
- `spotify-results.png` — evaluation metrics or example recommendations

Replace the placeholder paths in the post if you use different filenames.

**Large images:** The site displays images at a limited width (e.g. 280px for the model figure), but the browser still downloads the full file. To reduce file size and speed up loading, resize or compress the PNG (e.g. [Squoosh](https://squoosh.app/), or `sips -Z 560 /path/to/spotify-model.png` on Mac to max 560px width).
